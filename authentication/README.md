# Nextjs Authentication

> Authentication with Next.js.

## _Installation_

1. Clone or download this repository.

2. Create .env.local file and set the following enviroment variables

   ```sh
   MONGODB_URL=your_mongodb_url
   ```

3. run:

   ```sh
   $ npm install
   $ npm run dev
   ```
